# DATA221-Assignment-1

# Assignment 1 - Introduction to Data Science

This repository contains solutions for Assignment 1 of DATA 221.

Each question is implemented in a separate Python file and all files run without errors.

## Files Description

question1.py  
Controlled multiplication loop that multiplies consecutive integers until the product exceeds a threshold.

question2.py  
Creates a nested dictionary from a list of strings showing length and parity (even or odd).

question3.py  
Computes powers safely from a list of (x, y) pairs while skipping negative exponents.

question4.py  
Generates random values, sorts them, and finds indices where values are greater than or equal to x.

question5.py  
Validates circle radii, computes areas, and returns percentage coverage.

question6.py  
Performs distribution analysis and returns cumulative percentages.

question7.py  
Converts seconds since midnight into formatted time with AM/PM.

question8.py  
Creates a Pandas DataFrame and adds a computed column.


## How to Run

Make sure Python (and pandas for Question 8) is installed.

Run any file using:

python question1.py
python question2.py
...
